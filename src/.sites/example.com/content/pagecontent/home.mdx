# Welcome

This is an example site built with the multi-site blogging platform.

## Getting Started

Explore the site to see what's possible:

- **[Blog](/blog)** - Read posts and articles
- **[About](/about)** - Learn more about this site
- **[Test Page](/test)** - See feature demonstrations

## Features

This platform supports:

- Multiple sites from a single codebase
- Markdown and MDX content
- Custom components per site
- Dynamic theming (light/dark mode)
- Blog with categories and tags
- Full-text search via Pagefind
- Keystatic CMS for content editing
