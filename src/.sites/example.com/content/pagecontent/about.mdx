# About Example Site

Welcome to the **Example Site**! This is a minimal demonstration of the blog platform built with [Astro](https://astro.build/).

## Features

This example site showcases:

* Custom navigation menu (left and right)
* Blog posts with dates
* Draft posts
* Custom Astro components
* MDX content support

## Technology Stack

* **Astro** - Static site generator
* **Tailwind CSS** - Styling
* **MDX** - Enhanced markdown
* **Keystatic** - Content management (in development mode)

## About This Demo

This site was created as a minimal example to demonstrate the platform's capabilities and serve as a template for new sites.
